# Sales Prediction Using Python

## Objective
This project predicts product sales based on advertising budgets spent on TV, Radio, and Newspaper using Machine Learning.

## Technologies Used
- Python
- Pandas
- Matplotlib
- Scikit-learn

## Dataset
Advertising.csv

## Algorithm
Linear Regression

## Steps
1. Load the dataset.
2. Split the data into training and testing sets.
3. Train the Linear Regression model.
4. Predict sales.
5. Evaluate the model using R2 Score and MAE.

## Output
The model predicts sales based on advertising budgets and displays the performance metrics and graph.
